# Flow - Student Collaboration Tool 📝

Your go-to collaborative document editor designed specifically for students!

## Features ✨

Flow is a modern, Google Docs-like collaborative document editor built with students in mind. It includes all the essential features you need for document creation and editing:

### Text Formatting
- **Font Family Selection** - Choose from 7 different fonts (Arial, Times New Roman, Georgia, Courier New, Verdana, Comic Sans MS, Inter)
- **Font Sizes** - 7 different size options from Small to Maximum
- **Text Styling** - Bold, Italic, Underline, and Strikethrough
- **Text Color** - Full color picker for text color
- **Highlighting** - Highlight important text with custom colors

### Document Organization
- **Text Alignment** - Left, Center, Right, and Justify alignment
- **Line Spacing** - Choose from Single, 1.15, 1.5, Double, 2.5, or Triple spacing
- **Lists** - Create numbered (ordered) and bullet (unordered) lists
- **Clear Formatting** - Remove all formatting with one click

### Productivity Features
- **Undo/Redo** - Easily undo or redo your changes
- **Keyboard Shortcuts** - 
  - Ctrl/Cmd + B for Bold
  - Ctrl/Cmd + I for Italic
  - Ctrl/Cmd + U for Underline
  - Ctrl/Cmd + S to Save
- **Auto-save** - Your work is automatically saved every 3 seconds
- **Word & Character Count** - Live statistics at the bottom
- **Document Title** - Name your documents
- **Export** - Download your document as an HTML file

### User-Friendly Design
- Beautiful gradient interface with purple theme
- Responsive design that works on desktop and mobile
- Clean and intuitive toolbar
- Distraction-free writing experience
- Status bar with document statistics

## How to Use 🚀

1. **Open the Application**
   - Simply open `index.html` in your web browser
   - No installation or server required!

2. **Start Writing**
   - Click on the editor area and start typing
   - Your document is automatically saved to your browser's local storage

3. **Format Your Text**
   - Use the toolbar at the top to format your text
   - Select text and click formatting buttons
   - Choose fonts, sizes, and colors from the dropdowns

4. **Save & Export**
   - Your work auto-saves every 3 seconds
   - Click "Save" to manually save
   - Click "Export" to download your document as an HTML file

5. **Give Your Document a Title**
   - Click on "Untitled Document" at the top to rename

## Technical Details 🔧

- **Pure HTML, CSS, and JavaScript** - No frameworks required
- **Local Storage** - Documents are saved in your browser
- **Responsive Design** - Works on all screen sizes
- **Print Support** - Documents are print-friendly
- **No Backend Required** - Runs entirely in the browser
